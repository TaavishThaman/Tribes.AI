#Steps to run the panel
1) Install node.js
2) Install nodemon
3) Install all dependencies (npm i)
4) run command nodemon app.js
5) Server will start running on localhost:5000

#Tech stack used
1) Node.js
2) Jquery
3) Javascript
4) HTML/CSS/Bootstrap
5) Charts.js
